# RestaugrandVueJS
## Liste de points remarquable de notre projet
- Notre projet utilise la bibliothèque vuetify et ses composants comme par exemple la dataTable qui affiche les restaurants.
- Notre projet est responsive.
- Notre projet est découpé en composants distincts, ce qui rend notre code modulaire.
- Nous avons une page dynamique qui affiche les détails d'un restaurant à partir de son ID (présent dans l'URL de la page).
- Nous pouvons modifier un restaurant depûis le client.
- Notre projet présente des menus et des plats. Nous n'avons malheureusement pas eu le temps de réaliser un découpage plus propre concernant les données ou de créer un nouvel endpoint.
- Nous utilisons un système de theming pour les couleurs
